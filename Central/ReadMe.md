### Build Instructions

## Required
-  Node Version 18 LTS
-  NPM Version 9.5.1

## For Development
```bash
npm install
npm run dev
```

### for tests,

```bash
npm install
npm run test
npm run coverage
```

### for Production build

```bash
npm install
npm run build
```

- code will be built into ES5 compliant JS inside `ui/dist` folder, dist folder need to be deployed.
